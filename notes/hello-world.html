Java 第一课：Hello World

2026-05-09

什么是 Java

Java 是一门面向对象的编程语言，由 Sun Microsystems 于 1995 年发布。 它的口号是 "Write Once, Run Anywhere"（一次编写，到处运行）， 意思是编译好的 Java 程序可以在任何安装了 JVM（Java 虚拟机）的设备上运行。

JDK 的安装

编写 Java 程序前，需要先安装 JDK（Java Development Kit），它包含了编译和运行 Java 程序所需的工具。

  1. 前往 Oracle 官网下载 JDK（推荐 JDK 17 或 JDK 21 等 LTS 版本）
  2. 安装完成后，打开终端（CMD 或 PowerShell），输入以下命令验证安装：
java -version
javac -version

如果看到版本号输出，说明安装成功。

第一个 Java 程序

创建一个名为 HelloWorld.java 的文件，写入以下代码：

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

代码解释

  • public class HelloWorld — 定义一个公开的类，类名必须和文件名一致。
  • public static void main(String[] args) — 程序的入口方法，Java 程序从这里开始执行。
    • public：任何地方都能访问
    • static：不需要创建对象就可以调用
    • void：这个方法没有返回值
    • String[] args：接收命令行参数
  • System.out.println() — 向控制台打印一行文字并换行。

编译 & 运行

在终端中进入文件所在目录，依次执行：

javac HelloWorld.java    ← 编译，生成 HelloWorld.class
java HelloWorld          ← 运行，输出结果

运行结果：

Hello, World!

小结

这是 Java 学习的第一课。我们了解了 Java 的跨平台特性，安装了 JDK 开发环境， 并编写、编译、运行了第一个 Java 程序。虽然只有一行输出，但它包含了 Java 程序最核心的结构： 类、main 方法和输出语句。万丈高楼平地起，从今天开始，一步一个脚印 💪
