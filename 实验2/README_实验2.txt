==================================================
  Java Swing 扫雷游戏 —— 实验2 提交版
==================================================

尊敬的老师，您好！

本程序是一个基于 Java Swing 编写的高级扫雷游戏，
请您按以下方式运行：

  【运行方法】
  直接双击 "双击运行扫雷.bat" 即可启动游戏。
  (前提：电脑已安装 JDK 且 java 命令在 PATH 环境变量中)

==================================================
  程序亮点功能
==================================================

1. 良好的 GUI 界面
   - 雷区数字采用彩色渲染（1-蓝色, 2-绿色, 3-红色, 4+-紫色），
     视觉效果清晰美观。
   - 支持右键插旗标记（🚩），踩雷后显示炸弹标记（💣）。

2. 可调节难度
   - 支持"容易 (9×9, 10雷)"与"困难 (16×16, 40雷)"两种模式。
   - 通过顶部下拉框即可动态切换，雷区自动刷新。

3. 计时与计步
   - 顶部状态栏实时显示游戏用时（秒）与有效操作步数。
   - 首次点击开始计时，游戏结束时停止。

4. 进度保存与读取
   - 点击"保存进度"按钮，游戏状态（雷图、翻开状态、
     插旗标记、计时、步数）序列化为 minesweeper.sav 文件。
   - 点击"读取进度"按钮，完美恢复游戏现场，继续未完成的挑战。
