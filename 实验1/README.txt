尊敬的用户 / 老师：

一、如何运行本软件
--------------------
方式一 (推荐)：直接双击 "双击运行.bat" 文件，程序将自动启动。
方式二：在命令行/终端中切换到本目录，执行以下命令：
    java ArithmeticQuizApp

注意：请确保您的电脑已安装 Java 运行环境 (JRE/JDK 8 及以上版本)。

二、核心功能说明（实验1 实现内容）
--------------------
本软件是一个基于 Java Swing 的图形界面算术练习程序，主要功能包括：

1. 三级难度选择
   - 容易：10以内加减法
   - 提高：50以内加减乘
   - 挑战：100以内加减乘除

2. 智能随机出题
   - 随机生成操作数与运算符
   - 自动避免减法出现负数（适合小学生）
   - 除法确保能整除，避免复杂小数

3. 实时答案判断与反馈
   - 输入答案后点击"确定提交"进行验证
   - 答对显示绿色恭喜提示
   - 答错显示红色提示并给出正确答案

4. 得分统计
   - 实时统计答题总数与正确数
   - 切换难度自动清零得分

5. 友好的图形界面
   - 大字号公式显示，适合小学生观看
   - 浅蓝色配色方案，界面美观
   - 按钮防重复提交保护

6. 交互细节
   - 输入框自动获得焦点
   - 输入格式合法性检查
   - 空答案提醒
